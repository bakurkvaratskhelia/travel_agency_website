# Travel Agency Website

A modern, responsive **Travel Agency Website** built with HTML, CSS, JavaScript, and contemporary web best practices.  
This project is designed to showcase clean web design, user-friendly interface, and an engaging experience to help customers explore and book their dream vacations.

---

## Features

- Beautiful home/landing page
- Gallery of featured destinations and tours
- Detailed trip pages
- Booking/request forms
- Contact information section
- Responsive design for desktop, tablet, and mobile devices
- Smooth navigation and interactive elements
- (Optional) Testimonials, FAQs, or blog/articles section

## Getting Started

1. **Clone the repo**  
   ```bash
   git clone https://github.com/bakurkvaratskhelia/travel_agency_website.git
   cd travel_agency_website
   ```

2. **Open `index.html` in your browser**  
   No build tools required. All static files ready-to-go.
